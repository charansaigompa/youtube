A YouTube-like frontend application built using React, Redux, and Tailwind CSS that replicates core UI and interaction behavior of YouTube, including video listing, watch page, live chat UI, comments, and search suggestions.

This project focuses on real-world frontend architecture, state management, and user interaction handling.

 Features

 # Home Page

Video cards with thumbnail, title, channel info

Responsive grid layout

# Search

Search bar with live suggestions

Keyboard & mouse interaction handling

Clean UX similar to YouTube

# Watch Page

Embedded video player

Video details section

Comments section

Suggested videos panel

# Live Chat (UI simulation)

Real-time message rendering

Auto-scroll behavior

Redux-managed chat state

# Sidebar

Toggleable sidebar using global Redux state

YouTube-style navigation layout

# Shimmer / Loading UI

Skeleton loaders for better UX during data fetch

# Tech Stack

React – Component-based UI

Redux Toolkit – Global state management

React Router DOM – Client-side routing

Tailwind CSS – Styling & responsive design

YouTube Data API – Video data

Vite – Fast development build tool

# Key Learnings & Concepts Used

React Hooks (useState, useEffect, useRef)

Redux global vs local state management

Conditional rendering & loading states

Handling focus, blur, and keyboard events

Preventing stale Redux data issues

Component reusability & clean architecture

Debouncing & UX optimizations
